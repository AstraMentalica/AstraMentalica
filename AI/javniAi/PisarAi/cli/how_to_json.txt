🚀 KONČNA NAVODILA
1. Ustvari mapo
bash
mkdir -p /podatki/ai
chmod 755 /podatki/ai
2. Kopiraj datoteki
ai_deepseek_json.php → SISTEM/includes/

ai_deepseek_json.php (admin) → SISTEM/admin/

3. Dodaj konstanto v config
php
define('PODATKI_PATH', ROOT_PATH . '/podatki');
4. Vnesi svoj DeepSeek API ključ
Odpri admin vmesnik

V kartici "API ključi" klikni "Dodaj"

Vnesi svoj ključ

5. Uživaj! 🎉
