# Modul CodexAntiqua

Codex Antiqua — starodavni rokopisi in kodeksi iz vseh tradicij sveta.

## Navodila
Modul je ustvarjen po standardni predlogi AstraMentalica.  
Vstopna točka: `modul.php`  
Konfiguracija: `modul.json`

## Datoteke
- `modul.php` — vstopna točka, akcije `info` in `domov`
- `modul_codexantiqua.php` — glavna klasa `ModulCodexAntiqua`
- `modul_codexantiqua_funkcije.php` — pomožne funkcije
- `modul_codexantiqua_pravila.php` — dostopna pravila
- `modul_codexantiqua_jsonbaza.php` — branje/pisanje `modul.json`
- `modul_codexantiqua_api.php` — AI/API integracija
- `modul_codexantiqua_config.php` — lokalna konfiguracija
- `modul_codexantiqua_demo.php` — demo prikaz
- `podatki/` — manifesti in metapodatki
