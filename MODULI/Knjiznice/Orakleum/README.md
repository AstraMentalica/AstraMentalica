# Modul Orakleum

Orakleum — orakeljna knjižnica. Prerokbe, divinacija in sistemi vedeževanja.

## Navodila
Modul je ustvarjen po standardni predlogi AstraMentalica.  
Vstopna točka: `modul.php`  
Konfiguracija: `modul.json`

## Datoteke
- `modul.php` — vstopna točka, akcije `info` in `domov`
- `modul_orakleum.php` — glavna klasa `ModulOrakleum`
- `modul_orakleum_funkcije.php` — pomožne funkcije
- `modul_orakleum_pravila.php` — dostopna pravila
- `modul_orakleum_jsonbaza.php` — branje/pisanje `modul.json`
- `modul_orakleum_api.php` — AI/API integracija
- `modul_orakleum_config.php` — lokalna konfiguracija
- `modul_orakleum_demo.php` — demo prikaz
- `podatki/` — manifesti in metapodatki
