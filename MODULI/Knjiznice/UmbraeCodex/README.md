# Modul UmbraeCodex

Umbrae Codex — skrita knjiga senc. Ezoterika, okultna besedila in skrivno znanje.

## Navodila
Modul je ustvarjen po standardni predlogi AstraMentalica.  
Vstopna točka: `modul.php`  
Konfiguracija: `modul.json`

## Datoteke
- `modul.php` — vstopna točka, akcije `info` in `domov`
- `modul_umbraecodex.php` — glavna klasa `ModulUmbraeCodex`
- `modul_umbraecodex_funkcije.php` — pomožne funkcije
- `modul_umbraecodex_pravila.php` — dostopna pravila
- `modul_umbraecodex_jsonbaza.php` — branje/pisanje `modul.json`
- `modul_umbraecodex_api.php` — AI/API integracija
- `modul_umbraecodex_config.php` — lokalna konfiguracija
- `modul_umbraecodex_demo.php` — demo prikaz
- `podatki/` — manifesti in metapodatki
