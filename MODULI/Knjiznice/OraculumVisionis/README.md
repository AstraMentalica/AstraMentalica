# Modul OraculumVisionis

Oraculum Visionis — knjiga videnj in prerokovanj. Vizionarska besedila, sanje in prerokbe.

## Navodila
Modul je ustvarjen po standardni predlogi AstraMentalica.  
Vstopna točka: `modul.php`  
Konfiguracija: `modul.json`

## Datoteke
- `modul.php` — vstopna točka, akcije `info` in `domov`
- `modul_oraculumvisionis.php` — glavna klasa `ModulOraculumVisionis`
- `modul_oraculumvisionis_funkcije.php` — pomožne funkcije
- `modul_oraculumvisionis_pravila.php` — dostopna pravila
- `modul_oraculumvisionis_jsonbaza.php` — branje/pisanje `modul.json`
- `modul_oraculumvisionis_api.php` — AI/API integracija
- `modul_oraculumvisionis_config.php` — lokalna konfiguracija
- `modul_oraculumvisionis_demo.php` — demo prikaz
- `podatki/` — manifesti in metapodatki
