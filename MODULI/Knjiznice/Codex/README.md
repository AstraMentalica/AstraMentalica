# Modul Codex

Codex — osrednja knjižnica duhovnih besedil in komentarjev. Jedro znanja sistema.

## Navodila
Modul je ustvarjen po standardni predlogi AstraMentalica.  
Vstopna točka: `modul.php`  
Konfiguracija: `modul.json`

## Datoteke
- `modul.php` — vstopna točka, akcije `info` in `domov`
- `modul_codex.php` — glavna klasa `ModulCodex`
- `modul_codex_funkcije.php` — pomožne funkcije
- `modul_codex_pravila.php` — dostopna pravila
- `modul_codex_jsonbaza.php` — branje/pisanje `modul.json`
- `modul_codex_api.php` — AI/API integracija
- `modul_codex_config.php` — lokalna konfiguracija
- `modul_codex_demo.php` — demo prikaz
- `podatki/` — manifesti in metapodatki
