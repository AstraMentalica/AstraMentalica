# Modul Avalon

Avalon — keltska mistična tradicija arthurijanske legende. Otok jabolk, vitezi in sveti gral.

## Navodila
Modul je ustvarjen po standardni predlogi AstraMentalica.  
Vstopna točka: `modul.php`  
Konfiguracija: `modul.json`

## Datoteke
- `modul.php` — vstopna točka, akcije `info` in `domov`
- `modul_avalon.php` — glavna klasa `ModulAvalon`
- `modul_avalon_funkcije.php` — pomožne funkcije
- `modul_avalon_pravila.php` — dostopna pravila
- `modul_avalon_jsonbaza.php` — branje/pisanje `modul.json`
- `modul_avalon_api.php` — AI/API integracija
- `modul_avalon_config.php` — lokalna konfiguracija
- `modul_avalon_demo.php` — demo prikaz
- `podatki/` — manifesti in metapodatki
