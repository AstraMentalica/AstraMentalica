# Modul Druidica

Druidica — modrost druidov. Sveta drevesa, naravni cikli, keltski obredi in kozmologija.

## Navodila
Modul je ustvarjen po standardni predlogi AstraMentalica.  
Vstopna točka: `modul.php`  
Konfiguracija: `modul.json`

## Datoteke
- `modul.php` — vstopna točka, akcije `info` in `domov`
- `modul_druidica.php` — glavna klasa `ModulDruidica`
- `modul_druidica_funkcije.php` — pomožne funkcije
- `modul_druidica_pravila.php` — dostopna pravila
- `modul_druidica_jsonbaza.php` — branje/pisanje `modul.json`
- `modul_druidica_api.php` — AI/API integracija
- `modul_druidica_config.php` — lokalna konfiguracija
- `modul_druidica_demo.php` — demo prikaz
- `podatki/` — manifesti in metapodatki
