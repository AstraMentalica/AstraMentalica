# Modul Ogham

Ogham — starodavna keltska pisava in orakelj dreves. Vsakemu drevesu svoja runa, vsaki runi svoja moč.

## Navodila
Modul je ustvarjen po standardni predlogi AstraMentalica.  
Vstopna točka: `modul.php`  
Konfiguracija: `modul.json`

## Datoteke
- `modul.php` — vstopna točka, akcije `info` in `domov`
- `modul_ogham.php` — glavna klasa `ModulOgham`
- `modul_ogham_funkcije.php` — pomožne funkcije
- `modul_ogham_pravila.php` — dostopna pravila
- `modul_ogham_jsonbaza.php` — branje/pisanje `modul.json`
- `modul_ogham_api.php` — AI/API integracija
- `modul_ogham_config.php` — lokalna konfiguracija
- `modul_ogham_demo.php` — demo prikaz
- `podatki/` — manifesti in metapodatki
