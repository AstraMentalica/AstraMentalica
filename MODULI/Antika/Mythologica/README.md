# Modul Mythologica

Mythologica — zbirka mitov in legend antičnega sveta. Heroji, pošasti in kozmogonske zgodbe.

## Navodila
Modul je ustvarjen po standardni predlogi AstraMentalica.  
Vstopna točka: `modul.php`  
Konfiguracija: `modul.json`

## Datoteke
- `modul.php` — vstopna točka, akcije `info` in `domov`
- `modul_mythologica.php` — glavna klasa `ModulMythologica`
- `modul_mythologica_funkcije.php` — pomožne funkcije
- `modul_mythologica_pravila.php` — dostopna pravila
- `modul_mythologica_jsonbaza.php` — branje/pisanje `modul.json`
- `modul_mythologica_api.php` — AI/API integracija
- `modul_mythologica_config.php` — lokalna konfiguracija
- `modul_mythologica_demo.php` — demo prikaz
- `podatki/` — manifesti in metapodatki
