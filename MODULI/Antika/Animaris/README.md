# Modul Animaris

Animaris — duhovnost narave in živali v antičnem svetu. Živalski totemi, svete živali in mistični bestiarij.

## Navodila
Modul je ustvarjen po standardni predlogi AstraMentalica.  
Vstopna točka: `modul.php`  
Konfiguracija: `modul.json`

## Datoteke
- `modul.php` — vstopna točka, akcije `info` in `domov`
- `modul_animaris.php` — glavna klasa `ModulAnimaris`
- `modul_animaris_funkcije.php` — pomožne funkcije
- `modul_animaris_pravila.php` — dostopna pravila
- `modul_animaris_jsonbaza.php` — branje/pisanje `modul.json`
- `modul_animaris_api.php` — AI/API integracija
- `modul_animaris_config.php` — lokalna konfiguracija
- `modul_animaris_demo.php` — demo prikaz
- `podatki/` — manifesti in metapodatki
