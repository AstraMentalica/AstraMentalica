# Modul Labyrinthus

Labyrinthus — skrivnosti antičnih labirintov in iniciacijskih poti. Simbolika, miti in duhovni pomen.

## Navodila
Modul je ustvarjen po standardni predlogi AstraMentalica.  
Vstopna točka: `modul.php`  
Konfiguracija: `modul.json`

## Datoteke
- `modul.php` — vstopna točka, akcije `info` in `domov`
- `modul_labyrinthus.php` — glavna klasa `ModulLabyrinthus`
- `modul_labyrinthus_funkcije.php` — pomožne funkcije
- `modul_labyrinthus_pravila.php` — dostopna pravila
- `modul_labyrinthus_jsonbaza.php` — branje/pisanje `modul.json`
- `modul_labyrinthus_api.php` — AI/API integracija
- `modul_labyrinthus_config.php` — lokalna konfiguracija
- `modul_labyrinthus_demo.php` — demo prikaz
- `podatki/` — manifesti in metapodatki
