# Modul Devorum

Devorum — panteon antičnih božanstev. Grški, rimski in orientalski bogovi, njihovi atributi in miti.

## Navodila
Modul je ustvarjen po standardni predlogi AstraMentalica.  
Vstopna točka: `modul.php`  
Konfiguracija: `modul.json`

## Datoteke
- `modul.php` — vstopna točka, akcije `info` in `domov`
- `modul_devorum.php` — glavna klasa `ModulDevorum`
- `modul_devorum_funkcije.php` — pomožne funkcije
- `modul_devorum_pravila.php` — dostopna pravila
- `modul_devorum_jsonbaza.php` — branje/pisanje `modul.json`
- `modul_devorum_api.php` — AI/API integracija
- `modul_devorum_config.php` — lokalna konfiguracija
- `modul_devorum_demo.php` — demo prikaz
- `podatki/` — manifesti in metapodatki
