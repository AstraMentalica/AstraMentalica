# Modul Shamanica

Šamanizem — poti med svetovi. Bobnanje, rastlinski učitelji in šamanska tradicija.

## Navodila
Modul je ustvarjen po standardni predlogi AstraMentalica.  
Vstopna točka: `modul.php`  
Konfiguracija: `modul.json`

## Datoteke
- `modul.php` — vstopna točka, akcije `info` in `domov`
- `modul_shamanica.php` — glavna klasa `ModulShamanica`
- `modul_shamanica_funkcije.php` — pomožne funkcije
- `modul_shamanica_pravila.php` — dostopna pravila
- `modul_shamanica_jsonbaza.php` — branje/pisanje `modul.json`
- `modul_shamanica_api.php` — AI/API integracija
- `modul_shamanica_config.php` — lokalna konfiguracija
- `modul_shamanica_demo.php` — demo prikaz
- `podatki/` — manifesti in metapodatki
