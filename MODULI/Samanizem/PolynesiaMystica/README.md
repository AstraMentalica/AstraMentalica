# Modul PolynesiaMystica

Polynesia Mystica — polinezijska modrost. Mana, navigacija zvezd in duhovna tradicija.

## Navodila
Modul je ustvarjen po standardni predlogi AstraMentalica.  
Vstopna točka: `modul.php`  
Konfiguracija: `modul.json`

## Datoteke
- `modul.php` — vstopna točka, akcije `info` in `domov`
- `modul_polynesiamystica.php` — glavna klasa `ModulPolynesiaMystica`
- `modul_polynesiamystica_funkcije.php` — pomožne funkcije
- `modul_polynesiamystica_pravila.php` — dostopna pravila
- `modul_polynesiamystica_jsonbaza.php` — branje/pisanje `modul.json`
- `modul_polynesiamystica_api.php` — AI/API integracija
- `modul_polynesiamystica_config.php` — lokalna konfiguracija
- `modul_polynesiamystica_demo.php` — demo prikaz
- `podatki/` — manifesti in metapodatki
