# Modul AfricanaMystica

Africana Mystica — afriške duhovne tradicije. Ubuntu, animizem, predniški duhovi.

## Navodila
Modul je ustvarjen po standardni predlogi AstraMentalica.  
Vstopna točka: `modul.php`  
Konfiguracija: `modul.json`

## Datoteke
- `modul.php` — vstopna točka, akcije `info` in `domov`
- `modul_africanamystica.php` — glavna klasa `ModulAfricanaMystica`
- `modul_africanamystica_funkcije.php` — pomožne funkcije
- `modul_africanamystica_pravila.php` — dostopna pravila
- `modul_africanamystica_jsonbaza.php` — branje/pisanje `modul.json`
- `modul_africanamystica_api.php` — AI/API integracija
- `modul_africanamystica_config.php` — lokalna konfiguracija
- `modul_africanamystica_demo.php` — demo prikaz
- `podatki/` — manifesti in metapodatki
